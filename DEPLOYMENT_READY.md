```bash
git push origin main
```

**Step 3: Enable GitHub Pages**
1. Go to Repository Settings
2. Click "Pages" (left sidebar)
3. Select "Deploy from branch"
4. Choose "main" branch
5. Choose "/root" folder
6. Click Save

**Step 4: Wait for Deployment**

- Wait 1-2 minutes
